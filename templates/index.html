Frequently Asked Questions

What type of PDF files are supported?

The tool supports digitally generated PDF files with selectable text. Examples include bank statements, invoices, GST reports, ledger reports, and other business documents.

Does this tool support scanned PDFs or images?

No. Currently, scanned PDFs, image-based documents, screenshots, and handwritten files are not supported because OCR functionality is not available yet.

Will my table formatting remain the same in Excel?

The tool tries to preserve the original table structure, including rows, columns, and headers. However, some complex layouts may need small manual corrections.

Is my uploaded data safe?

Yes. Your files are processed securely, are not shared with anyone, and are automatically removed after processing.

What should I do if the converted Excel file is not accurate?

Try uploading a cleaner PDF with properly aligned tables and selectable text. If the PDF contains complex formatting, you may need to make small edits in Excel.

Can I upload multiple-page PDFs?

Yes. The tool supports PDFs with multiple pages and combines extracted data into one Excel file wherever possible.

Does the tool support password-protected PDFs?

No. Password-protected or encrypted PDF files must be unlocked before uploading.

Who can use this tool?

This tool is useful for accountants, finance teams, CA firms, auditors, tax consultants, business owners, and anyone who regularly works with PDF reports.

Is there any file size limit?

Large files may take more time to process. For better speed and accuracy, it is recommended to upload smaller or properly structured PDF files whenever possible.
